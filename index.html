Результати виконання лабораторної роботи

Інструкція: Всі результати роботи методів (drive(), trip(), info()) та виклики статичних функцій виводяться у консоль браузера.

Для перегляду результатів:

  1. Натисніть F12 або праву кнопку миші → Переглянути код (Inspect).
  2. Перейдіть на вкладку Console.
  3. Оновіть сторінку (F5).

Виконані завдання:

  • Створення об'єктів через new Object() та літерал.
  • Робота з конструкторами (ES5) та прототипами.
  • Реалізація ієрархії класів (ES6): Square → Rectangle → Parallelogram.
  • Використання замикань та деструктуризації.
